## Installation

1. Install the `@kobalte/core` component from kobalte:

```bash
pnpm add @kobalte/core
```

2. Copy and paste the following code into your project.

<ComponentSource src="/src/components/ui/textfield.tsx" />

## Usage

```tsx
import { TextFieldRoot, TextFieldInput } from "~/components"
```

```tsx
<TextFieldRoot>
	<TextFieldInput type="email" placeholder="Email" />
</TextFieldRoot>
```
