## Installation

1. Install the `@kobalte/core` component from kobalte:

```bash
pnpm add @kobalte/core
```

2. Copy and paste the following code into your project.

<ComponentSource src="/src/components/ui/alert.tsx" />

## Usage

```tsx
import { Alert, AlertDescription, AlertTitle } from "~/components"
```

```tsx
<Alert>
	<i class="i-luicde:terminal" />
	<AlertTitle>Heads up!</AlertTitle>
	<AlertDescription>
		You can not add components and dependencies to your app using the cli,
		yet.
	</AlertDescription>
</Alert>
```
