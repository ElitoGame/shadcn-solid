## Manual Installation

<Steps>

### Add UnoCSS

Components are styled using UnoCSS. You need to install it in your project.

[Follow the UnoCSS installation instructions to get started.](https://unocss.dev/integrations/)

### Add dependencies

Add the following dependencies to your project:

```bash
pnpm add class-variance-authority
```

### Add icon library

This project uses [UnoCSS preset icon](https://unocss.dev/presets/icons) and [Iconify lucide](https://icon-sets.iconify.design/lucide/), but you can use any icon library.

### Path Aliases

I'm use the `~` alias to make it easier to import your components. This is how you can configure it in `tsconfig.json`:

```json title="tsconfig.json"
{
    "compilerOptions": {
        "baseUrl": "./",
        "paths": {
            "~/*": ["./src/*"]
        }
    }
}
```

If you prefer to use a different alias than `~`, you'll need to update the `import` statements when adding components.

### Import styles to your app

Import the styles to your root:

```tsx title="src/root.tsx"
import "@unocss/reset/tailwind.css"
import "uno.css"
```

If you not using UnoCSS preflights:

```tsx title="src/root.tsx"
import "@unocss/reset/tailwind.css"
import "uno.css"
import "~/styles/global.css"
```

You can now start adding components to your project.
